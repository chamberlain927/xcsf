void pop_del ( XCSF *  xcsf,
SET *  kset 
)

Deletes a single classifier from the population set.

Parameters
xcsfThe XCSF data structure.
ksetA set to store deleted macro-classifiers for later memory removal.

Selects two classifiers using roulete wheel selection with the deletion vote and deletes the one with the largest condition + prediction length. For fixed-length representations this is the same as one roulete spin.
